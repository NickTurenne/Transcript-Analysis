# Transcript Performance Analysis

This project analyzes academic performance across terms and departments using
credit-weighted GPA metrics and comparisons to class averages.

## Highlights
- GPA trends over time compared to class averages
- Department-level performance analysis
- Difficulty index for course-level normalization
- Clean separation of ingestion, cleaning, analysis, and visualization
